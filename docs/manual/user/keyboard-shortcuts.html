Keyboard Shortcuts

Speed up your workflow with keyboard shortcuts for navigation, calling, and common actions.

Overview

The Aspendora Connect application supports keyboard shortcuts that let you navigate, manage calls, and interact with the interface without reaching for your mouse. Learning a few key shortcuts can significantly speed up your daily workflow.

Shortcuts are available whenever the application is focused. They are disabled when you are typing in a text field, search box, or form input so they do not interfere with your typing.

Quick Reference
Press ? at any time in the application to open the keyboard shortcuts help dialog. This shows all available shortcuts for the current page.

Modifier Keys

Shortcuts use modifier keys that differ between operating systems. Throughout this guide, Cmd/Ctrl means:

  • macOS — Use the Cmd (Command) key
  • Windows / Linux — Use the Ctrl (Control) key

Global Shortcuts

These shortcuts work from anywhere in the application, regardless of which page you are on.

Shortcut Action
Cmd/Ctrl + K Open the search / command palette. Type to search for pages, extensions, contacts, or actions. Coming Soon
Escape Close the current modal, dialog, or dropdown. If the softphone panel is open, this closes it.
Cmd/Ctrl + S Save the current form. Works on any page with a form (profile settings, voicemail greeting, etc.).
? Show the keyboard shortcuts help dialog. Displays all shortcuts available on the current page.

Navigation Shortcuts

Navigation shortcuts use a two-key sequence: press G first, then the second key. For example, to go to the dashboard, press G then D (press and release G, then press D).

Shortcut Action
G then D Go to the Dashboard
G then E Go to the Extensions page (your extension and the directory)
G then V Go to your Voicemail inbox
G then H Go to Call History
G then P Go to your Profile settings
Two-Key Sequences
Navigation shortcuts are sequential, not simultaneous. Press G and release it, then press the second key within about one second. If you wait too long between keys, the sequence resets and nothing happens.

Table & List Shortcuts

When viewing tables or lists (call history, voicemail, directory), these shortcuts help you navigate and interact with items without using the mouse.

Shortcut Action
J Move the selection down to the next row in the table or list.
K Move the selection up to the previous row in the table or list.
Enter Open the currently selected item. For example, open a voicemail message to play it, or open call details.
/ Focus the search or filter input for the current list. Start typing to filter results.
N Create a new item on list pages where applicable (for example, start a new call from the call history page).

Tips for Table Navigation

  • The currently selected row is highlighted with a subtle background color so you can see which item is focused.
  • When you first navigate to a list page, no row is selected. Press J to select the first row.
  • You can combine these shortcuts: press / to search, type your filter, press Escape to leave the search box, then use J/K to navigate the filtered results.
  • Press Escape to deselect the current row and return focus to the page.

Softphone Shortcuts

These shortcuts control the web softphone. They use modifier keys so they do not conflict with normal typing.

Shortcut Action
Cmd/Ctrl + Shift + C Toggle the softphone panel open or closed. When opened, the dial pad is focused and ready for input.
Cmd/Ctrl + Shift + A Answer an incoming call. Only works when there is an active incoming call ringing on the softphone.
Cmd/Ctrl + Shift + H Hang up the current call. Works during an active call to end it immediately.
Softphone Must Be Registered
Softphone shortcuts only work when the softphone is connected (green status indicator). If the softphone is disconnected, the answer and hang up shortcuts will have no effect. See the Web Softphone page for connection troubleshooting.

Notes

  • Keyboard shortcuts are disabled when you are focused on a text input, search field, or form element to prevent conflicts with typing.
  • Some shortcuts may conflict with browser shortcuts or operating system shortcuts. If a shortcut does not seem to work, check if your browser or OS is intercepting it.
  • Shortcuts are not customizable at this time. Custom key bindings may be available in a future release.
  • Screen reader users: shortcuts are designed to work alongside common screen reader key combinations. If you experience conflicts, you may need to toggle your screen reader's pass-through mode.
