Roadmap

All planned features from the original roadmap have been implemented. This page describes potential future enhancements.

Overview

All major features on the Aspendora Connect roadmap have been delivered and are available in the current release. The platform now includes full support for integrations (Teams, Zendesk, Slack), omnichannel messaging (WhatsApp, Messenger, Email), billing and usage metering, STIR/SHAKEN caller ID verification, HA/DR infrastructure, migration tools, developer APIs with webhooks, plugin marketplace, hospitality module, receptionist console, and more.

Feature Requests
If you have specific feature requests or want to influence future priorities, contact your MSP administrator. Feature requests are tracked and prioritized based on demand across the tenant base.

Future Possibilities

While all planned features have been delivered, the platform continues to evolve. Areas under consideration for future releases include:

  • Advanced AI capabilities — Enhanced real-time agent assist, automated quality scoring, predictive analytics
  • Additional CRM integrations — HubSpot, Zoho, ServiceNow, and other CRM platforms
  • Advanced workforce optimization — ML-based schedule prediction, real-time adherence monitoring
  • Video conferencing — WebRTC-based video calling with screen sharing
  • Fax-over-IP — T.38 fax support with digital archiving
  • Advanced IVR — Visual IVR builder with NLU-based routing
