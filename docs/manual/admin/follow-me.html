Follow Me

Forward unanswered calls to cell phones and external numbers using sequential or simultaneous ring strategies, with per-destination ring times.

Overview

Follow Me extends an extension's reach beyond the desk phone. When enabled, calls that are not answered at the extension within a configurable time are automatically forwarded to one or more external numbers — typically a cell phone, home office line, or alternate work number. This ensures that mobile workers, field staff, and on-call personnel never miss important calls.

Follow Me is configured per extension and supports up to 10 external destinations. Each destination has its own independent ring time, and the system can ring them one at a time (sequential) or all at once (simultaneous).

Follow Me vs Call Forwarding
Follow Me and Call Forwarding (CF) are related but distinct features. Call Forwarding immediately redirects all calls to a single destination without ringing the extension first. Follow Me tries the extension first (optionally), then falls through to external destinations. Follow Me also supports multiple destinations and ring strategies. See the comparison section below for a detailed breakdown.

Configuring Follow Me

Follow Me is an update operation on an existing extension. Each extension has a Follow Me configuration that is disabled by default. To enable it, submit a PUT request to the extension's Follow Me endpoint. 

PUT /api/v1/extensions/{extension_id}/follow-me

{
  "enabled": true,
  "strategy": "SEQUENTIAL",
  "ring_extension_first": true,
  "extension_ring_time": 25,
  "destinations": [
    { "destination": "+15551234567", "ring_time": 20 },
    { "destination": "+15559876543", "ring_time": 25 }
  ]
}

strategy enum No SEQUENTIAL Ring strategy: SEQUENTIAL (one at a time) or SIMULTANEOUS (all at once)

extension_ring_time int No 25 Seconds to ring the desk extension before moving to external destinations (5–120). Only applies when ring_extension_first is true.

Field Type Required Default Description

Ring Strategies

SEQUENTIAL

In sequential mode, the system rings each destination one at a time, in list order. If the first destination does not answer within its ring_time, the system hangs up that leg and dials the next destination. This continues until a destination answers or all destinations have been tried.

  • Destinations are tried in the order they appear in the destinations list
  • Each destination rings for its own ring_time independently
  • If no destination answers, the call falls through to the extension's voicemail (if configured)
  • Total ring time is the sum of all ring times (extension + each destination)

SIMULTANEOUS

In simultaneous mode, the system rings all external destinations at the same time. The first destination to answer gets the call; all other ringing legs are cancelled. This minimizes the total time the caller waits before someone picks up.

  • All destinations ring at the same moment
  • First to answer wins — other legs are torn down
  • Ring time is determined by the longest ring_time among all destinations
  • If no destination answers before all ring times expire, the call falls through to voicemail
Which Strategy to Choose
Use sequential when you have a priority order (try the office cell first, then the personal cell, then the home phone). Use simultaneous when you want the fastest possible answer — whichever phone the user is near will pick up.

Ring Extension First

When ring_extension_first is true (the default), the system rings the user's desk extension for extension_ring_time seconds before engaging any external destinations. This gives the user a chance to answer at their desk before the call goes to their cell phone.

When set to false, the system skips the desk extension entirely and immediately rings the external destinations. This is useful for users who are never at their desk (field workers, remote employees) and do not want the caller to wait through a desk ring that will never be answered.

Setting Behavior
ring_extension_first: true Ring desk phone for extension_ring_time seconds, then try external destinations
ring_extension_first: false Skip desk phone entirely, immediately ring external destinations

Destination Types

Follow Me destinations are external phone numbers dialed through the outbound SIP trunk. Any number that can be reached via the platform's outbound routes can be a Follow Me destination.

  • Cell phones — The most common destination. Forward to the user's mobile number so they can take calls on the go.
  • Home office lines — Landline or VoIP number at a home office for remote workers.
  • Alternate work numbers — A second office or a shared workspace phone.
  • Hotel/travel numbers — Temporary numbers when traveling, updated as needed.
  • International numbers — Forward to international destinations (subject to outbound route permissions and toll charges).
Toll Charges
Follow Me calls to external numbers are placed as outbound calls through your SIP trunk. Standard per-minute charges apply based on your trunk provider's rate. High-volume Follow Me usage to mobile or long-distance numbers can significantly increase your monthly telecom costs. Monitor CDRs for Follow Me call volume.

Ring Time Per Destination

Each destination has its own ring_time setting (default 20 seconds) that controls how long the system rings that number before giving up. This allows fine-tuned control over the calling experience.

  • Short ring time (5–15s) — For destinations where the user is unlikely to answer (backup numbers). Fails fast and moves on.
  • Standard ring time (20–30s) — Typical for cell phones. Gives the user time to hear the ring and pick up.
  • Long ring time (30–120s) — For destinations where the user may need extra time (remote locations, shared phones).
Cell Phone Voicemail Interception
If a Follow Me destination is a cell phone, the carrier's voicemail system may answer the call before the ring time expires. Most cell carriers send calls to voicemail after 15–25 seconds. Set the Follow Me ring_time shorter than the carrier's voicemail pickup time to prevent the cell voicemail from intercepting the call. A ring_time of 15–18 seconds is typically safe for most carriers.

Follow Me vs Call Forwarding

Both features forward calls to external numbers, but they serve different purposes and behave differently. The table below compares the two.

Feature Follow Me Call Forwarding (CF)
Rings extension first Optional (configurable) No — immediately forwards
Multiple destinations Up to 10 Single destination
Ring strategy Sequential or simultaneous Not applicable (single target)
Per-destination ring time Yes (5–120s each) Not applicable
Trigger No answer at extension (after ring time) Unconditional, busy, or no-answer (variant-dependent)
User self-service Configured by admin or user Often toggled by users via feature codes
Best for Mobile workers who need multi-hop failover Simple "send all calls to my cell" redirection

Common Use Cases

Mobile Worker

A sales representative who is frequently out of the office. Ring the desk phone first briefly, then try the cell phone. Keep ring times short to avoid the cell carrier's voicemail intercepting the call. 

{
  "enabled": true,
  "strategy": "SEQUENTIAL",
  "ring_extension_first": true,
  "extension_ring_time": 15,
  "destinations": [
    { "destination": "+15551234567", "ring_time": 18 }
  ]
}

On-Call Rotation

An IT support extension where the on-call person changes weekly. Update the Follow Me destinations each week to point to the current on-call technician's cell phone. Skip the desk extension since nobody sits at the on-call desk. 

{
  "enabled": true,
  "strategy": "SEQUENTIAL",
  "ring_extension_first": false,
  "destinations": [
    { "destination": "+15557771000", "ring_time": 20 },
    { "destination": "+15557772000", "ring_time": 20 }
  ]
}

Executive with Multiple Devices

Ring the desk phone and all devices simultaneously. Whichever device the executive is near picks up first. 

{
  "enabled": true,
  "strategy": "SIMULTANEOUS",
  "ring_extension_first": true,
  "extension_ring_time": 30,
  "destinations": [
    { "destination": "+15558001000", "ring_time": 25 },
    { "destination": "+15558002000", "ring_time": 25 }
  ]
}

Fully Remote Employee

A remote worker with no desk phone. Skip the extension ring entirely and go straight to their home office number, falling back to their cell phone. 

{
  "enabled": true,
  "strategy": "SEQUENTIAL",
  "ring_extension_first": false,
  "destinations": [
    { "destination": "+15553001000", "ring_time": 25 },
    { "destination": "+15553002000", "ring_time": 20 }
  ]
}
